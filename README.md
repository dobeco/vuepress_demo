
# 文档库、博客

本仓库用来存放前端学习笔记，以前做笔记散落在电脑的各个角落，每次翻看笔记比较麻烦，有个集中管理的地方比较好。

`vuepress`是个不错的选择，带搜索功能，查找方便，单页`SSR`应用，加载这种纯文字的网页非常快！



主题: [vuepress-theme-reco](https://vuepress-theme-reco.recoluan.com/)
